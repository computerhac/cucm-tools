Step 1 — Source Gateway

Migrates an SCCP analog gateway (e.g. VG224) or an MGCP analog gateway to a CUCM-controlled SIP analog gateway (VG410/VG420/VG450). Because the SIP analog endpoint does not support shared lines, each port is deprovisioned on the source and immediately re-provisioned on the target — one at a time or all at once.

How do I find the chassis MAC?
On the gateway CLI:
VG2xx (SCCP):  show ccm-manager  or  show interface FastEthernet 0/0 | include address
VG4xx (SIP):   show platform | include MAC  or  show interfaces GigabitEthernet 0/0/0 | include Hardware
Recommended before migrating — captures every port's device + line + chassis bind so a restore can undo any deprovision.
